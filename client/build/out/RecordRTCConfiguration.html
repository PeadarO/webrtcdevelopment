Class: RecordRTCConfiguration

RecordRTCConfiguration

new RecordRTCConfiguration(mediaStream, config)

It configures the 2nd parameter passed over RecordRTC and returns a valid "config" object.

RecordRTCConfiguration is an inner/private helper for RecordRTC.
Parameters:
Name Type Description
mediaStream MediaStream MediaStream object fetched using getUserMedia API or generated using captureStreamUntilEnded or WebAudio API.
config object {type:"video", disableLogs: true, numberOfAudioChannels: 1, bufferSize: 0, sampleRate: 0, video: HTMLVideoElement, getNativeBlob:true, etc.}
