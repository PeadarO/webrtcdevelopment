# stats 


getStats() psrt of w3c Webrtc API, used to collect metrics from a WebRTC session. 
Its attribites tell about the health of media streams in the session.

Returned reports include : 

googTrack
googLibjingleSession
googCertificate
googComponent
googCandidatePair
localCandidate
remoteCandidate
